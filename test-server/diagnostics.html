SDK Diagnostics Test

How to Use This Page

  1. Initialize SDK: Click the "Initialize SDK" button to call amplitude.init(). This uses a backdoor to set the diagnostics sample rate to 1 (100%), ensuring all diagnostics events are tracked.
  2. Add Plugin: Click the "Add Plugin (Remove User/Device ID)" button to add a plugin that removes user_id and device_id from events. This mimics dropped events and triggers diagnostics tracking.
  3. View Diagnostics: Open DevTools → Application → IndexedDB to see diagnostics metrics tracked and saved in the database.
